# 🤖 Bybit Futures Bot — RSI + MA Combo

Торговый бот для Bybit Futures с комбинированной стратегией RSI + скользящие средние.

---

## 📁 Структура файлов

```
bybit_bot/
├── bot.py                # Главный файл бота
├── config.py             # Настройки (API ключи, параметры)
├── backtest.py           # Модуль бэктестинга
├── telegram_notifier.py  # Telegram уведомления
├── requirements.txt      # Зависимости
└── bot.log               # Лог-файл (создаётся автоматически)
```

---

## ⚙️ Установка

```bash
pip install -r requirements.txt
```

---

## 🔑 Настройка

Откройте `config.example.py` и заполните:

```python
API_KEY    = "ваш_api_key_с_bybit"
API_SECRET = "ваш_api_secret_с_bybit"
TESTNET    = True   # Начните с True!

TELEGRAM_TOKEN   = "токен_от_@BotFather"
TELEGRAM_CHAT_ID = "ваш_chat_id"
```

## 📊 Стратегия

### Вход в LONG:
- Быстрая MA (20) пересекает медленную MA (50) снизу вверх (Golden Cross)
- **RSI < 55** (покупаем на откатах, не ждем экстремальной перепроданности)
- Сигнал берется по **закрытой свече** (подтверждение тренда)

### Вход в SHORT:
- Быстрая MA (20) пересекает медленную MA (50) сверху вниз (Death Cross)
- **RSI > 45** (продаем на откатах, не ждем экстремальной перекупленности)
- Сигнал берется по **закрытой свече**

### Выход из позиции:
- **Take-profit:** +3% (TP = 0.03)
- **Stop-loss:** -1% (SL = 0.01)  
- **Смена сигнала:** закрытие позиции при противоположном сигнале
